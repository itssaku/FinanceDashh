# 💰 Finance Dashboard UI

## 📌 Overview

This project is a **Finance Dashboard UI** built using **React (JSX), JavaScript, HTML, and CSS**. It provides users with a clean, interactive, and responsive interface to track and understand their financial activities.

The focus of this project is to demonstrate strong **frontend development skills**, including UI/UX design, state management, and interactive features.

---

## 🚀 Features

### 📊 Dashboard Overview

* Displays key financial metrics:

  * Total Balance
  * Total Income
  * Total Expenses
* Includes:

  * 📈 Time-based visualization (balance trends)
  * 🧾 Category-based visualization (spending breakdown)

---

### 📁 Transactions Section

* Displays transaction details:

  * Date
  * Amount
  * Category
  * Type (Income / Expense)
* Features:

  * 🔍 Search functionality
  * 🔃 Sorting
  * 🎯 Filtering by category/type

---

### 👥 Role-Based UI (Frontend Simulation)

* **Viewer** → Can only view data
* **Admin** → Can add/edit transactions
* Role switching using dropdown/toggle

---

### 📈 Insights Section

* Provides insights such as:

  * Highest spending category
  * Monthly comparison
  * Spending patterns

---

### 📱 Mobile Responsive Design

* Fully responsive UI for all screen sizes
* 📱 Optimized layout for mobile devices
* 🔽 **Bottom Navigation Bar (Mobile Only)** for better usability
* Smooth adaptability across tablets and desktops

---

### 🌙 Dark Mode

* Toggle between **Light & Dark mode**
* Improved readability and modern UI experience

---

### 💾 Data Persistence

* Uses **Local Storage** to:

  * Save transactions
  * Preserve user data
  * Maintain state even after page refresh

---

### ✨ Animations & Transitions

* Smooth UI interactions
* Subtle animations for:

  * Component rendering
  * Button interactions
  * Data updates

---

### 📤 Export Functionality

* Export financial data as:

  * 📄 CSV
  * 📄 JSON
* Useful for analysis and record keeping

---

### 🧠 State Management

* Managed using React Hooks:

  * `useState`
  * `useEffect`
* Handles:

  * Transactions
  * Filters & search
  * Role-based UI
  * Theme (Dark/Light mode)

---

### 🎨 UI & UX

* Clean and modern interface
* User-friendly navigation
* Handles edge cases:

  * No data available
  * Empty transaction list

---

## 🛠️ Tech Stack

* **Frontend:** React (JSX)
* **Languages:** JavaScript, HTML, CSS
* **Styling:** Custom CSS
* **State Management:** React Hooks
* **Storage:** Local Storage

## ⚙️ Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/finance-dashboard.git
   ```

2. Navigate to the project folder:

   ```bash
   cd finance-dashboard
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the project:

   ```bash
   npm start
   ```

5. Open in browser:

   ```
   http://localhost:3000
   ```
